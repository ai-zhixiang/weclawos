## 二、文件访问方式（全部走 AI 指令）

### ① 读取系统文件/相册

```
用户: "帮我找上个月拍的照片"

→ AI 理解 → App 用 ContentResolver 查询 MediaStore
   WHERE date BETWEEN '2026-06-01' AND '2026-06-30'
   AND mime_type LIKE 'image/%'
→ 结果返回 AI → AI 展示缩略图列表
→ 用户: "第三张"
→ AI 打开/处理/分享
```

### ② 接收外部 App 分享

```
用户在微信收到文件
→ 点「用其他应用打开」
→ 选择 WeClaw
→ App 接收 → AI 自动识别
→ "收到一份文件：产品需求文档.pdf
   需要我做什么？"
```

### ③ 拍照

```
用户: "拍张产品照片"

→ AI 调系统相机 Intent
→ 拍照完成 → 照片自动传给 AI
→ AI 处理/分析
```
