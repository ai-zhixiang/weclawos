# WeClaw 手机端 v2.0 设计方案

> 基于桌面版 HTML UI 设计语言优化
> 核心原则：整个 App = 一个带系统能力的聊天界面

---

## 一、页面架构

**1 个页面，1 个输入框，0 个 Tab。**

```
┌──────────────────────────────────────┐
│                                      │
│             消息列表                   │
│      (AI 对话 + 系统反馈)              │
│                                      │
│  ┌──────────────────────────┐        │
│  │ 你的账号: 138****0526    │        │
│  │ 版本: v0.1.0             │ ← AI   │
│  │ 缓存: 156MB              │        │
│  │ [清理缓存]               │        │
│  └──────────────────────────┘        │
│                                      │
├──────────────────────────────────────┤
│  [ 输入消息...      ] 🎤 📎 ➤       │  ← 唯一交互入口
└──────────────────────────────────────┘
```

## 二、设计语言（继承桌面版 HTML）

```kotlin
// Color.kt
object WeClawColors {
    val background = Color(0xFF0A0A0C)      // 主背景
    val surface = Color(0xFF1A1A2E)         // 卡片/消息气泡
    val surfaceVariant = Color(0xFF1E1E32)  // 输入栏背景
    val primary = Color(0xFF4A6CF7)         // 品牌蓝
    val accent = Color(0xFFF97316)          // 语音激活橙
    val textPrimary = Color(0xFFE6E6E6)     // 主文字
    val textSecondary = Color(0xFF808080)   // 副文字
    val border = Color(0xFF1E1E32)          // 边框
    val success = Color(0xFF4ADE80)         // 连接状态绿
}
```

## 三、系统能力通道（由 AI 触发）

| 能力 | 实现 |
|------|------|
| 📷 拍照 | `ACTION_IMAGE_CAPTURE` |
| 🖼️ 选照片 | `PickVisualMedia` |
| 📄 选文件 | `OpenDocument` |
| 📍 位置 | `FusedLocationProviderClient` |
| 🌐 链接 | WebView |
| 📤 分享 | `ACTION_SEND` |
| ⚙️ 设置 | AI 对话管理 |
