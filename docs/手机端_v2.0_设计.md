# WeClaw 手机端 v2.0 设计方案

> 基于现有 Android 项目重构，新增文件管理、相册、底部导航
> 设计语言继承桌面版 HTML UI 的暗色风格

## 二、对话页 (ChatScreen) — 升级

### 当前问题

| 问题 | 改进方案 |
|------|----------|
| 无语音输入按钮 | 输入框旁边加 🎤 按钮，集成 SpeechRecognizer |
| 无表情/快捷回复 | 底部加快捷操作栏（同桌面版：🎵🎬📖📰✍️） |
| 消息样式平淡 | 用 markdown 渲染回复（代码块、链接可点击） |

| 顶部栏浪费 | 减少高度，连接状态用圆点表示 |

## 六、UI 设计语言

统一继承桌面版 HTML UI 的设计风格：

```yaml
色板:
  background: "#0a0a0c"       # 主背景
  surface: "#1a1a2e"          # 卡片/表面
  surfaceVariant: "#1e1e32"   # 表面变体
  primary: "#4a6cf7"          # 品牌蓝
  primaryVariant: "#3b5de7"
  accent: "#f97316"           # 强调橙（语音激活色）
  textPrimary: "rgba(255,255,255,0.9)"
  textSecondary: "rgba(255,255,255,0.4)"
  border: "rgba(255,255,255,0.08)"
  
圆角: 16dp  # 卡片圆角
字体: "Noto Sans SC" / "PingFang SC"
动效: 0.3s ease  # 过渡动画
```

顶部状态栏：半透明 + blur，文字浅灰 `(0.4)`
输入框：毛玻璃效果（background blur）
消息气泡：圆角 16dp + 浅色背景
