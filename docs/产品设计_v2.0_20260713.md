## 二、桌面版定位

微侠桌面 = **家用 AI 影音终端**
- 开车就进全屏 AI 界面，两块屏：AI 对话 + 全屏播放
- 语音全控：听歌、看视频、有声书、查信息
- AI 代劳：用户只管说，浏览器/搜索/文件操作全部 AI 干
- 不要传统桌面、不要文件管理器、不要键鼠优先

## 三、技术方案

### 底座
| 组件 | 选型 | 理由 |
|------|------|------|
| 系统 | Alpine Linux 3.21 | 极轻（~200MB），稳定 |
| 内核 | linux-lts + linux-firmware | 兼容笔记本/台式机硬件 |
| 合成器 | Cage | 只跑一个全屏窗口，无多余功能 |
| 界面 | Chromium kiosk + 自定义 HTML UI | 视觉自由，Web Speech API 原生语音 |
| 音频 | PulseAudio + ALSA | 内置麦克风/音响兼容 |
| 后端 AI | Hermes Agent（WebSocket） | 已有 |

### 核心功能
| 功能 | 实现方式 |
|------|---------|
| 🎵 听歌 | HTML5 Audio / mpv 后端 |
| 🎬 看视频 | HTML5 Video / mpv 后端 |
| 📖 有声书 | Web Speech API / edge-tts |
| 🤖 信息查询 | AI 搜索（Hermes） |
| ☁️ 云端同步 | 微侠云端 API，手机互通 |

## 五、三版统一设计原则

1. **语音优先** — 所有核心操作都能用语音完成
2. **同一套 HTML UI** — 桌面版 = 浏览器 kiosk，TV 版 = WebView，手机版 = WebView
3. **云端互通** — 文件/照片/待办跨端同步
4. **AI 代操作** — 不提供传统文件管理/浏览器界面，全部交给 AI
