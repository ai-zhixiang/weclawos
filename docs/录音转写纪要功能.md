# WeClaw 录音 + 转写 + 会议纪要

## 用户场景

```
用户: "帮我记一下会议"

→ App: 开始录音，显示波形+计时
→ 用户说话...
→ 用户: "结束"
→ App: 停止录音 → 自动转写文字
→ 文字发给 AI
→ AI: 输出会议纪要（时间、议题、结论、待办）
→ 用户确认 → 保存
```

## 技术方案

### Android 端

| 组件 | 说明 |
|------|------|
| `MediaRecorder` | 录音，输出 PCM/WAV |
| `VoiceManager`（已有） | 实时转写（SpeechRecognizer） |
| `RecorderScreen.kt` 🆕 | 录音全屏界面 |
| `TranscriptionService.kt` 🆕 | 转写服务 |

### 云端 AI

| 步骤 | 说明 |
|------|------|
| ① 录音文件上传 | Multipart upload → 服务器 |
| ② ASR 转写 | 服务器调用 ASR API（Whisper/飞桨） |
| ③ 纪要生成 | AI 总结转写文本 |
| ④ 返回结果 | Markdown 会议纪要 |
